Design of my website

Purpose of my website

To showcase my work as a Graphic Designer.

Intended Audience

Mainly potential employers in fields related to information design, editorial and publishing, or branding.

Tone of Written Content

A concise, approchable tone that however is well-versed and conceptual.

Primary Content

The primary content will include a descripcion of myself as a designer and some "work" categories.

Secondary Content

Each category in "Work" will include pictures and descriptions for each selected project. The categories would be: Commercial, Arts and Culture, Social and Typographic. The main menu would also include a link to a Contact page where the user can find my Resume and e-mail address.

Navitagion

As users enter the site, they will have the title “Mariana Bernardez”, a brief description of myself and a menu with the categories of work outlined above. The clickable elements would be my name (which would direct the user to a “Contact” page) and the work categories, which when clicked take the user to a second level of navigation where thumbnails and names of the works would appear. In this second level a menu with “Home” and “Contact” would appear on the top right corner of the web page. The third level of navigation would be when clicking on such thumbnails, where the user would get more detailed information on that project (a description + pictures).

Landing Page - Level 1

Work Categories (4 in total) - Level 2

Project Page - Level 3

Contact Page - Level 2 or 3

Includes e-mail and Resume for download.
